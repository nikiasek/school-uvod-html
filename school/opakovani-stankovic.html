Sémantický zdrojový kód (X)HTML

Než začneme pracovat s CSS, musíme si být jisti, že jsme schopni adekvátně strukturovat dokumenty (X)HTML moderním způsobem. Opakem k tomu, co jste se naučili v předchozích letech webového vývoje, platí, že jazyk (X)HTML by neměl definovat vzhled a prezentaci webové stránky( to je práce pro CSS). Namísto toho je účelem (X)HTML poskytnout obsahu dokumentu kontext a význam - pak se nazývá sémantickým zdrojovým kódem.

Vytváříme sémanticky bohatý zdrojový kód (X)HTML

Nyní, když jsme pochopili výhody sémantického zdrojového kódu, je na místě otázka, jak se vypořádat s vytvářením kódu (X)HTML, jenž je co nejlépe strukturovaný a nese co nejvíce významu. Začneme používáním správných elementů (X)HTML. Pokud jde o odstavec, použijete značku <p>. Pokud jde o nadpis, použijte značku <hx>. Jestliže jde o seznam s odkazy, použijte značku <a> uvnitř elementu seznamu. K určení významu obsahu byste měli používat elementy jako jsou em a strong, a poté použít CSS k určení požadovaného vzhledu těchto elementů(jako je podtržení, kurzíva nebo tučnost)

Vytváříme šablonu stylů

Syntaxe CSS je poměrně "jednoduchá" potýkáme se zde pouze se seznamem pravidel. Jednoduchá šablona stylů může vypadat nějak takto: 

        
    h1 {
        color:blue;
    }
         
    h2 {
        color:green;
    }

Tato dvě pravidla uložte do textového souboru, ten pojmenujte s koncovkou .css a máte vlastní, plně validní (ačkoli velmi jednoduchou) externí šablonu stylů.
